# E-Store - Premium Electronics ở Store

Ứng dụng web thương mại điện tử hiện đại được xây dựng từ thiết kế Figma với React + Vite.

## Cấu trúc dự án

```
estore-web/
├── client/          # Frontend React application
└── server/          # Backend Node.js API (sẽ được thêm sau)
```

## 🚀 Bắt đầu nhanh

### Cài đặt

```bash
# Cài đặt dependencies cho frontend
cd client
npm install
```

### Chạy ứng dụng

```bash
# Development mode
npm run dev
```

Ứng dụng sẽ chạy tại `http://localhost:5173`
